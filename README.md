# Репозиторий дополнения CarVector Bot для Home Assistant

Этот репозиторий нужно залить на GitHub и добавить в Home Assistant как пользовательский репозиторий дополнений.

## Структура

- **repository.yaml** — описание репозитория для HA (обязательный файл в корне).
- **carvector_bot/** — одно дополнение (папка с именем = slug из config.yaml).

## Как добавить в Home Assistant

1. Создайте на GitHub новый репозиторий (например `carvector-bot-addon`). Репозиторий может быть **приватным** или публичным.
2. Заливайте в него **содержимое** папки `addons` (корень репозитория = папка `addons`):
   - в корне: `repository.yaml`, `README.md`;
   - папка `carvector_bot/` со всеми файлами внутри.
3. В Home Assistant: **Настройки** → **Дополнения** → **Магазин дополнений** → ⋮ → **Репозитории**.
4. Добавьте URL репозитория, например:  
   `https://github.com/Dolkir643/carvector-bot-addon`
5. Нажмите «Добавить», затем обновите список дополнений (или перезагрузите страницу).
6. В списке появится **CarVector Bot** — установите его, задайте конфигурацию и запустите.

## Важно

- В **repository.yaml** поле `url` должно совпадать с URL вашего репозитория на GitHub.
- Имя папки дополнения должно быть **carvector_bot** (с подчёркиванием), как в `slug` в config.yaml.
