# CarVector Bot — Home Assistant Add-on

Telegram-бот для поиска запчастей LAND ROVER на carvector.ru. Показывает цены, статусы поставщиков и оформление заявок.

## Установка

1. В Home Assistant: **Настройки** → **Дополнения** → **Магазин дополнений** → три точки (⋮) → **Репозитории**.
2. Добавьте репозиторий: `https://github.com/Dolkir643/carvector-bot-addon`
3. Обновите список (или перезагрузите страницу).
4. Найдите **CarVector Bot** в списке дополнений и нажмите **Установить**.

## Настройка

Во вкладке **Конфигурация** укажите:

- **bot_token** — токен от @BotFather
- **carvector_login** — email для входа на carvector.ru
- **carvector_password** — пароль
- **manager_chat_id** — ID чата для уведомлений о заявках (например 232066339)

Сохраните и запустите дополнение. Заявки сохраняются в `/data/orders.json` (том данных).
